## Описание игры

**Console RPG Adventure** — это консольная ролевая игра, в которой игрок управляет героем, исследует локации, сражается с врагами и получает опыт.

### Основные возможности

- Создание персонажа
- Сражения с монстрами
- Система уровней
- Инвентарь и предметы
- Исследование игрового мира

## Запуск проекта

Для запуска выполните следующие действия:

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/username/console-rpg.git
   ```

2. Перейдите в папку проекта:
   ```bash
   cd console-rpg
   ```

3. Запустите программу:
   ```bash
   python main.py
   ```

## Объяснение основных модулей

### `main.py`

Главная точка входа в приложение. Отвечает за запуск игрового цикла и взаимодействие пользователя с игрой.

### `player.py`

Содержит класс игрока:

- здоровье;
- опыт;
- уровень;
- инвентарь.

### `enemy.py`

Описывает врагов и их характеристики.

### `battle.py`

Реализует механику боя между игроком и противником.

### `inventory.py`

Управляет предметами и их использованием.
